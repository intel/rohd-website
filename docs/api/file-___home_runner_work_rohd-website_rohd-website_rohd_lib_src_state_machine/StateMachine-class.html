StateMachine

StateMachine<StateIdentifier> class

Simple class for FSM StateMachine.

Abstraction for representing Finite state machines (FSM). Contains the logic for performing the state transitions.

Constructors

StateMachine(dynamic clk, dynamic reset, StateIdentifier resetState, List<State<StateIdentifier>> _states)
Constructs a simple FSM, using the clk and reset signals. Also accepts the reset state to transition to resetState along with the List of _states of the FSM.
